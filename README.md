## Google sheet nodejs:
- install ``npm install``
- run ``npm run dev``

## Config
- rename ``.env.example`` to ``.env``. replace value with yours
- rename ``service-account.json.example`` to ``service-account.json``. replace value with yours
- edit config file: ``src/config.js``
